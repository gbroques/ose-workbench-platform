OSE Workbench Platform Documenation
===================================

A platform for workbench development by `Open Source Ecology <https://www.opensourceecology.org/>`_.

The following pages describe the standards and guidelines for implementing OSE Workbenches using the FreeCAD platform, including branching and workflow strategy, versioning, and patterns for solving common problems.

Every OSE workbench has the same structure and components to make working between various workbenches easier, and increase collaboration.
